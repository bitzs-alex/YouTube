Lists

HTML lists allow web developers to group a set of related items in lists.

In HTML there are three type of lists. Ordered, Unordered, and Definition List

Ordered Lists

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

  1. Item one
  2. Item two
  3. Item three

Unordered Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

Definition Lists

A definition list starts with the <dl> tag. Each definition term will be wrapped inside <dt> tag and the description will be sat inside <dd> tag.

Nesting Lists

Is formed when one list item, wraps another!
